1-¿Cuál es la diferencia principal entre position relative y position absolute?

La diferencia es que en position relative permite desplazar un elemento
y en la position absolute permite colocar un elemento en un lugar especifico.

2-¿Qué valor de position se utiliza por defecto para los elementos HTML?

Las propiedades que se utilizan son: top, bottom, left o right.

3-¿Qué propiedad se utiliza para establecer la capa de apilamiento de elementos superpuestos?

Para establecer la capa de apilamiento se utiliza Z-index

4-¿Cuándo se utiliza position fixed en CSS?

El elemento se mantendrá en su posición incluso si se desplaza la página.

5-¿Cómo se comporta un elemento con position sticky al desplazarse?

Un elemento con la position sticky se queda fijo

6-¿Cuál es el valor de position adecuado para un elemento que se desea que siga el flujo normal del documento?

El position adecuado que se utiliza para que siga el flujo normal del documento es "position static"

7-¿Cuál es la forma más común de mover un elemento con position relative?

La forma más común de mover un elemento con position relative es con: top,bottom,left,right

8-¿En qué casos se puede utilizar z-index para controlar el orden de apilamiento de elementos?

El z-index, lo podemos usar para cajas que están una encima de la otra, y lo queremos acomodar de otra forma
